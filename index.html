The Fifteen Puzzle

The fifteen puzzle is a simple game first invented around 1874. It consists of 15 blocks in a 4Ã—4 grid, each labeled with an integer from 1 to 15. The goal is to arrange the blocks sequentially from 1 to 15, with the empty block (numbered 16 if it were there) in the bottom right. Modern versions often replace the numbers with images, which form a complete picture when solved.

To start, the blocks are placed randomly in the grid. Some initial configurations are unsolvable, so be careful!
